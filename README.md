# Haratres FIT Staj Programı - Değerlendirme Ödevi

Bu proje, Haratres FIT Staj Programı kapsamında verilen değerlendirme ödevinin çözümlerini içermektedir.

## Soru 1 — Karakter Sayacı
Kullanıcının girdiği cümle içinde belirlenen bir karakterin kaç kez geçtiğini bulan program.
Büyük/küçük harf duyarlılığı seçeneği mevcuttur.

## Soru 2 — Ürün Yönetim Sistemi
Kullanıcıdan alınan ürün bilgilerini kaydeden, farklı kriterlere göre sıralayan ve sepet hesaplama yapan program.
İndirim mantığı: Sepetteki bir önceki üründen daha ucuz ürün geldiğinde önceki ürüne indirim uygulanır.

## Kullanılan Teknolojiler
- Java 17+
- IntelliJ IDEA

## Nasıl Çalıştırılır
1. Projeyi IntelliJ IDEA ile aç
2. Soru1.java veya Soru2.java dosyasını çalıştır
